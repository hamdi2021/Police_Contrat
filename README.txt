Application de Recherche Police-Contrat
===============================

Cette application permet de rechercher un numéro de Police et d'afficher le Contrat correspondant.

Installation
-----------
1. Assurez-vous que Python 3.6 ou supérieur est installé sur votre système.
2. Décompressez l'archive dans un dossier de votre choix.

Lancement de l'application
-------------------------
- Sous Windows: Double-cliquez sur le fichier "lancer_application.py" ou exécutez-le via la commande:
  python lancer_application.py
  
- Sous Linux/Mac: Ouvrez un terminal dans le dossier de l'application et exécutez:
  python3 lancer_application.py

Utilisation
----------
1. Entrez un numéro de Police dans le champ de recherche.
2. Cliquez sur le bouton "Rechercher" ou appuyez sur la touche Entrée.
3. Le Contrat correspondant s'affichera dans la zone de résultat.

Dépendances
----------
- Python 3.6+
- Tkinter (installé automatiquement avec Python sous Windows, peut nécessiter une installation séparée sous Linux/Mac)

En cas de problème
----------------
Si l'application ne démarre pas, vérifiez que Python et Tkinter sont correctement installés sur votre système.
